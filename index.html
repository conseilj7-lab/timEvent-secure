Comment ça marche ?
1
Cliquez "☁️ Sauvegarder maintenant"
Télécharge un fichier tim-event-data.json sur votre appareil
2
Sur iPhone : déplacez le fichier dans iCloud Drive
Ouvrez "Fichiers" → trouvez le fichier téléchargé → maintenez → "Déplacer" → iCloud Drive → créez un dossier "Tim Event"
3
Sur MacBook : l'accès est automatique
Finder → iCloud Drive → Tim Event → glissez le fichier dans l'app via "⬇️ Charger depuis iCloud"
4
Routine recommandée
Fin de journée : Sauvegarder → glisser dans iCloud. Début sur autre appareil : Charger depuis iCloud
📱 Installer sur iPhone

1. Ouvrez dans Safari
2. Bouton Partager ↑
3. "Sur l'écran d'accueil"
4. Ajouter

Important

Utilisez toujours le même navigateur sur chaque appareil pour conserver vos données localStorage. La sauvegarde iCloud est votre filet de sécurité en cas de changement d'appareil.
